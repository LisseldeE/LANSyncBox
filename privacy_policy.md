# LANSyncBox 隐私政策

最后更新日期：2026-06-22

## 引言

LANSyncBox（以下简称"本应用"）是一款局域网文件同步工具，致力于保护用户隐私。本隐私政策旨在说明本应用在信息收集、使用和保护方面的做法。

## 我们收集的信息

**本应用不收集任何用户的个人信息。**

- 本应用不要求注册账户
- 本应用不收集您的姓名、邮箱、地址、电话等个人身份信息
- 本应用不收集设备标识符、位置信息或其他设备信息

## 文件传输

- 本应用的文件同步功能**仅在局域网内进行**
- 文件传输直接发生在您授权的设备之间，**不经过任何中间服务器**
- 文件内容不会被上传到云端或任何第三方服务
- 传输过程使用您本地网络，应用开发者无法访问您的文件

## 网络请求

本应用仅在以下场景发起网络请求：

1. **检查更新功能**：当用户主动点击"检查更新"按钮时，应用会请求 GitHub 或 Gitee 的公开 API 获取最新版本号信息。此请求仅包含版本号查询，不包含任何用户身份信息。

除此之外，本应用不会主动发起任何网络请求。

## 本地存储

本应用在用户设备本地存储以下配置信息：

- **语言偏好设置**（中文/英文）
- **固定房间号设置**（用户主动启用时保存）

这些信息仅保存在应用所在目录的 `config.json` 文件中，**不会上传到任何服务器**，卸载应用或删除该文件即可清除。

## 权限说明

本应用运行所需的权限：

- **局域网访问权限**：用于设备间的文件同步和房间发现
- **文件系统访问权限**：用于读写用户选择的同步文件夹
- **网络访问权限**：仅用于检查更新功能

本应用不会请求访问您的摄像头、麦克风、通讯录、位置等敏感权限。

## 第三方服务

本应用不集成任何第三方分析、广告或追踪服务。

检查更新功能访问的 GitHub API 和 Gitee API 受各自平台的隐私政策约束：

- GitHub 隐私政策：https://docs.github.com/zh/site-policy/privacy-policies
- Gitee 隐私政策：https://gitee.com/help/articles/4186

## 儿童隐私

本应用不针对 13 岁以下儿童，也不会有意收集儿童的个人信息。

## 数据安全

由于本应用不收集任何用户数据，因此不存在数据泄露风险。您的文件仅在本地网络传输，建议您在使用时确保局域网环境安全。

## 隐私政策变更

如果本隐私政策发生变更，更新后的政策将随应用新版本一同发布，并在本页面注明更新日期。
