# LANSyncBox - 局域网文件实时同步工具

## 项目简介

LANSyncBox 是一款轻量级的局域网文件实时同步工具，实现多人隔空文件共享。在同一局域网内即可实现文件共享与同步，无需公网连接。采用全新QT6架构，支持大文件流式传输，优化多连接同步逻辑，提供流畅的用户体验。

## 功能特性

### 实时同步
- 文件添加、修改、删除、重命名操作实时同步至所有连接端
- 支持自定义6位数字房间号，方便记忆与分享
- 可选密码验证，确保同步安全
- 大文件流式传输，避免内存占用过高
- 并发传输限制（最多同时传输3个文件），优化系统资源占用
- 初次连接全量同步，自动对齐双端差异
- 加入房间双端版本号验证，确保连接兼容性

### 传输可靠性
- 传输取消机制：传输中途文件变化时自动取消，避免文件损坏
- 断点续传：大文件分块按索引定位写入，失败重发不损坏文件
- 完整性校验：文件接收完成时校验大小，不完整自动丢弃
- TCP 缓冲区优化：增大收发缓冲区，避免大文件背压超时
- 背压自适应：发送超时自动重试，区分取消与背压，确保大文件稳定传输
- 失败客户端自动重发：广播中某客户端失败时跳过并重发整个文件，确保最终同步

### 多端同步
- 主机端监控同步文件夹内所有变化
- 变更实时同步至所有连接端
- 接收连接端上传的文件，根据"隐藏标记"决定是否转发
- 支持文件夹同步，自动递归同步文件夹内所有文件
- 实时显示传输进度，支持进度条可视化

### 连接端功能
- 文件变更上传至主机端（不直接发给其他连接端）
- 支持"对外隐藏本机文件"开关
- 实时显示同步状态
- 断线重连后自动增量同步
- 实时进度显示，支持文件传输进度条

### 文件预览
- 支持双击文件直接预览，预览窗口为只读模式
- 避免预览时误修改文件内容

### 缓存管理
- 支持管理同步缓存，清理本地缓存文件
- 缓存路径独立管理，减少系统盘占用

### 多语言支持
- 支持中文/英文界面切换
- 实时切换语言无需重启程序
- 适配多语言环境

### 系统托盘
- 最小化窗口自动隐藏到系统托盘
- 托盘菜单显示当前房间号
- 双击托盘图标恢复窗口
- 点击"退出"关闭所有同步并退出程序

### 界面优化
- 全新QT6架构，界面更流畅
- 优化的界面交互逻辑，提升使用体验
- 实时进度显示，支持文件传输进度条
- 优化的文件列表显示逻辑

## 使用方法

### 主机端（创建连接）

1. 点击"创建连接"按钮
2. 输入自定义6位数字房间号（系统会检测是否已被占用）
3. 可选：设置密码保护
4. 选择需要同步的文件夹路径
5. 可选：开启"允许连接端互相同步"功能
6. 点击创建，进入同步状态界面

### 连接端（加入连接）

1. 点击"加入连接"按钮
2. 输入房间号
3. 如需密码，弹窗输入验证
4. 选择同步文件存放路径
5. 点击连接，自动从主机全量同步一次

## 同步逻辑

### 总体原则

文件重复等问题由系统资源管理器处理，同步只负责保持所有端文件一致。采用缓存文件、列表操作来处理同步内容，确保同步稳定可靠。

### 主机端

- 监控同步文件夹内所有变化（添加/修改/删除/重命名）
- 变更实时同步至所有连接端
- 接收连接端上传的文件，根据"隐藏标记"决定是否转发
- 支持并发传输限制（最多同时传输3个文件），避免资源占用过高
- 支持文件夹同步，自动递归同步文件夹内所有文件
- 大文件采用流式传输，避免内存占用过高
- 可查看所有同步记录及文件来源
- 可查看在线连接端列表

### 连接端

- 文件变更上传至主机端（不直接发给其他连接端）
- 支持"对外隐藏本机文件"开关
- 实时显示同步状态
- 断线重连后自动增量同步
- 实时进度显示，支持文件传输进度条

### 补充规则

- **冲突处理**：以最后修改时间较新的为准
- **主机离线**：所有连接端提示"房间已关闭"
- **传输协议**：TCP + 自定义协议
- **大文件处理**：采用流式分块传输，避免内存占用过高
- **并发控制**：最多同时传输3个文件，优化系统资源占用
- **传输取消**：文件变化时自动取消传输，发送 FILE_CANCEL 通知接收端清理
- **断点续传**：分块按 chunk_index 定位写入，失败重发不影响已接收部分
- **完整性校验**：FILE_END 时校验文件大小，不完整则丢弃临时文件
