# LANSyncBox

局域网文件同步工具

## 简介

LANSyncBox 是一款轻量级的局域网文件同步工具，支持多台电脑之间的实时文件同步。在同一局域网内即可实现文件共享与同步，无需公网连接。

## 功能特性

- **实时同步**：文件添加、修改、删除操作实时同步至所有连接端
- **自定义房间号**：支持自定义6位数字房间号，方便记忆与分享
- **密码保护**：可选密码验证，确保同步安全
- **隐藏模式**：支持"对外隐藏本机文件"，文件仅同步至主机
- **断线重连**：自动检测连接状态，断线后自动重连并增量同步
- **系统托盘**：最小化到托盘，不影响日常工作

## 使用方法

### 主机端（创建连接）

1. 点击"创建连接"按钮
2. 输入自定义6位数字房间号（系统会检测是否已被占用）
3. 可选：设置密码保护
4. 选择需要同步的文件夹路径
5. 可选：开启"允许连接端互相同步"功能
6. 点击创建，进入同步状态界面

### 连接端（加入连接）

1. 点击"加入连接"按钮
2. 输入房间号
3. 如需密码，弹窗输入验证
4. 选择同步文件存放路径
5. 点击连接，自动从主机全量同步一次

## 同步逻辑

### 总体原则

文件重复等问题由系统资源管理器处理，同步只负责保持所有端文件一致。

### 主机端

- 监控同步文件夹内所有变化（添加/修改/删除）
- 变更实时同步至所有连接端
- 接收连接端上传的文件，根据"隐藏标记"决定是否转发
- 支持最大同时传输限制（≤3个端），避免资源占用过高
- 可查看所有同步记录及文件来源
- 可查看在线连接端列表

### 连接端

- 文件变更上传至主机端（不直接发给其他连接端）
- 支持"对外隐藏本机文件"开关
- 实时显示同步状态
- 断线重连后自动增量同步

### 补充规则

- **冲突处理**：以最后修改时间较新的为准
- **主机离线**：所有连接端提示"房间已关闭"
- **传输协议**：TCP + 自定义协议

## 系统托盘

- 最小化窗口自动隐藏到系统托盘
- 托盘菜单显示当前房间号
- 双击托盘图标恢复窗口
- 点击"退出"关闭所有同步并退出程序
